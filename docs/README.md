# Docs

Use this page as the repo-local docs landing page.

Open these files in roughly this order:

- `../AGENTS.md` — repo-local startup and handoff contract
- `../scripts/README.md` — repo-local verification and helper script inventory
- `architecture.md` — runtime flow, package layout, dependencies, and non-goals
- `constraints.md` — hard sync-scope rules and boundary limits
- `observability.md` — logging, tracing, health expectations, and failure modes
- `cd/_index.md` — current CD-specific scope notes and repo-local authority
- `cd/change-request.md` — task template for future controller change requests
